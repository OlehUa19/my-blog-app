# My Blog App

## Опис
Цей веб-додаток дозволяє користувачам створювати, редагувати та видаляти пости. Кожен пост має заголовок, опис і автора. Всі пости зберігаються на сервері та відображаються на головній сторінці.

## Технології
- **Backend**: Node.js, Express.js
- **Frontend**: HTML, CSS, JavaScript
- **Зберігання даних**: в пам'яті (для простоти), можна розширити для збереження в базі даних.

## Як запустити

### Крок 1: Клонувати репозиторій
Клонуйте репозиторій на свій локальний комп'ютер:

```bash
git clone https://github.com/yourusername/my-blog-app.git

далі перейти в дерикторію проекту 

-cd project

потім встановіть всі необхідні пакети за допомогою npm

-npm install

Запустіть сервер

-npm start
